First-time setup

  1. Go to developer.yahoo.com/apps and create an app
  2. Set the Redirect URI to http://localhost:5000/auth/callback
  3. Enable the Fantasy Sports API permission (Read)
  4. Copy the Client ID and Client Secret into your config.txt file (see config.example.txt)
  5. Restart the server, then click Authenticate with Yahoo
